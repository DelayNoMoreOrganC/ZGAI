# 真实用户场景测试

## 场景1：律师新建案件

1. 登录系统
2. 点击"新建案件"
3. 填写案件信息
4. 点击"AI智能填充"上传文书
5. 保存案件
6. 在案件列表查看新案件
7. 点击案件名称进入详情
8. 切换各个Tab查看

## 场景2：律师使用AI问答

1. 进入知识库
2. 点击"AI知识问答"
3. 输入问题："民事诉讼时效是多久？"
4. 查看AI回答

## 场景3：律师查看客户

1. 在案件详情点击客户名称
2. 查看客户详情
3. 查看该客户的其他案件

## 场景4：切换视图

1. 在案件列表页
2. 点击"看板视图"
3. 查看案件按阶段分列
4. 点击"列表视图"切回
