## 问题击破过程

### 问题1：UserRepository查询错误

**症状**：
```json
{"code":500,"message":"系统内部错误: 用户不存在: 1"}
```

**根因分析**：
- `JwtAuthenticationFilter`将`userId`（Long）作为principal
- `DocGenerateController`调用`getName()`返回"1"
- 然后用`userRepository.findByUsername("1")`查询 → 找不到

**解决方案**：
创建`SecurityUtil`工具类，兼容两种principal类型：
```java
public Long getCurrentUserId() {
    Object principal = SecurityContextHolder
                          .getContext()
                          .getAuthentication()
                          .getPrincipal();

    if (principal instanceof Long) {
        return (Long) principal;  // 直接返回userId
    }

    String username = principal.toString();
    return userRepository.findByUsername(username)
            .map(User::getId)
            .orElseThrow(...);
}
```

**修复文件**：
- ✅ `SecurityUtil.java` - 新建通用工具类
- ✅ `DocGenerateController.java` - 更新使用SecurityUtil

## 关键代码

### SecurityUtil.java

```java
@Component
@RequiredArgsConstructor
public class SecurityUtil {
    private final UserRepository userRepository;

    public Long getCurrentUserId() {
        Object principal = SecurityContextHolder.getContext()
                                              .getAuthentication()
                                              .getPrincipal();

        if (principal instanceof Long) {
            return (Long) principal;
        }

        String username = principal.toString();
        return userRepository.findByUsername(username)
                .map(User::getId)
                .orElseThrow(() -> new RuntimeException("用户不存在: " + username));
    }
}
```

### DocGenerateController.java（修复后）

```java
@RestController
@RequestMapping("ai/generate-doc")
@RequiredArgsConstructor
public class DocGenerateController {
    private final DocGenerateService docGenerateService;
    private final SecurityUtil securityUtil;  // 使用工具类

    @PostMapping
    @PreAuthorize("hasAnyRole('ADMIN', 'LAWYER')")
    public Result<String> generateDocument(@RequestBody DocGenerateRequest request) {
        Long userId = securityUtil.getCurrentUserId();  // 简洁调用
        String result = docGenerateService.generateDocument(request, userId);
        return Result.success(result);
    }
}
```
