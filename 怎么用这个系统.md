## 一、如何访问系统

1. 打开浏览器
2. 输入地址：http://localhost:3017
3. 输入账号：admin

## 二、核心功能位置

### 1. 案件管理
- **案件列表**：登录后 → 左侧菜单"案件" → "案件列表"
- **新建案件**：案件列表页 → 右上角"新建案件"
- **案件详情**：案件列表 → 点击案件名称
- **编辑案件**：案件详情页 → 右上角"编辑"
- **归档库**：案件列表 → 左侧"归档库"
- **回收站**：案件列表 → 左侧"回收站"

### 2. 案件详情5个Tab
- **Tab1 基本案情**：案件详情 → 点击"基本案情"
- **Tab2 办案记录**：案件详情 → 点击"办案记录"
- **Tab3 受理单位**：案件详情 → 点击"受理单位"
  - 财产保全
  - 执行情况
  - 庭审记录
  - 承办人员
- **Tab4 案件文档**：案件详情 → 点击"案件文档"
- **Tab5 案件动态**：案件详情 → 点击"案件动态"

### 3. AI功能
- **AI OCR识别**：案件详情 → 案件文档Tab → 点击"AI识别"
- **AI文书生成**：案件详情页 → 点击"AI文书生成"
- **AI法律问答**：知识库 → "AI知识问答"

### 4. 知识库
- **知识库列表**：左侧菜单"知识库"
- **AI问答**：知识库 → "AI知识问答"
- **新建文章**：知识库列表 → "新建文章"

### 5. 客户管理
- **客户列表**：左侧菜单"客户"
- **新建客户**：客户列表 → "新建客户"
- **客户详情**：客户列表 → 点击客户名称

### 6. 其他功能
- **工作台**：左侧菜单"工作台"（登录后默认首页）
- **日程管理**：左侧菜单"日程"
- **财务管理**：左侧菜单"财务"
- **审批管理**：左侧菜单"审批"
- **行政OA**：左侧菜单"行政"
- **数据统计**：左侧菜单"统计"

## 四、批量操作

**案件列表页**：
1. 勾选多个案件（复选框）
2. 点击"批量操作"按钮
3. 选择操作：
   - 批量结案
   - 批量归档
   - 修改主办律师
   - 批量删除

## 五、常见问题

### Q1: 登录后看不到案件？
**A**: 系统初始化有2个案件。点击左侧"案件"→"案件列表"

### Q2: 点击"编辑"没反应？
**A**: 刷新页面（F5）

### Q3: AI问答没响应？
**A**: 检查Ollama是否运行（http://localhost:11434）

### Q4: 找不到某个功能？
**A**:
- 所有功能都在左侧菜单
- 案件详情有5个Tab（顶部切换）
- 点击"案件名称"进入详情
