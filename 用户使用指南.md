## 📋 核心功能使用指南

### 1. 案件管理

#### 1.1 查看案件列表
- 路径：登录后自动跳转到工作台，点击左侧菜单"案件管理"
- 功能：
  - 筛选：按类型/状态/等级/律师/法院筛选
  - 批量操作：选中多个案件后，点击"批量操作"
  - 视图切换：列表视图 ↔ 看板视图

#### 1.2 新建案件
- 点击右上角"新建案件"按钮
- 填写表单（A-F六个部分）：
  - A. 基本信息（案件类型、名称、案号等）
  - B. 当事人（点击"添加当事人"）
  - C. 代理律师费
  - D. 应收款（点击"添加应收款"）
  - E. 结案/归档信息（如适用）
  - F. 关联信息（关联客户、关联案件）
- 点击"AI智能填充"可自动识别法律文书
- 点击"查重"检查是否有重复案件
- 点击"提交"创建案件

#### 1.3 编辑案件 ⭐ (已修复)
- 在案件列表，点击案件名称进入详情
- 点击右上角"编辑"按钮
- 修改案件信息
- 点击"保存"完成编辑

#### 1.4 批量操作
- 在案件列表，勾选要操作的案件
- 点击"批量操作"下拉菜单
- 可选操作：
  - 批量结案
  - 批量归档
  - 修改主办律师
  - 批量删除

### 2. AI智能辅助

#### 2.1 AI文书识别（OCR）
- 路径：案件详情 → 案件文档Tab → 点击"AI识别"
- 功能：上传法律文书图片/PDF，自动提取案件信息
- 支持：起诉状、判决书、仲裁书等

#### 2.2 AI文书生成
- 路径：案件详情 → 点击"AI文书生成"
- 可生成：
  - 起诉状初稿
  - 答辩状初稿
  - 代理词框架
  - 法律意见书框架
- 选择文书类型 → 点击"生成" → 等待AI生成 → 编辑后使用

#### 2.3 AI法律问答 ⭐
- 路径：知识库 → AI知识问答
- 使用方法：
  - 输入法律问题（如："民事诉讼时效是多久？"）
  - 点击"提问"
  - 系统基于知识库+AI生成答案
  - 显示相关法律依据

**技术说明**：
- 当前使用：Ollama本地AI（qwen3:8b模型）
- 检索方式：TF-IDF关键词检索
- 响应时间：约3秒

## 🛠️ 常见问题

### Q1: 登录后看不到案件列表？
**A**: 检查是否正确登录。如果还是看不到，可能是数据为空，先创建一个测试案件。

### Q2: 点击"编辑"按钮没反应？
**A**: 这是刚修复的功能。刷新页面重试。如果还不行，检查浏览器控制台是否有错误。

### Q3: AI问答没有响应？
**A**: 检查Ollama是否运行：
```bash
curl http://localhost:11434/api/tags
```
如果没有返回模型列表，需要启动Ollama。

### Q4: 批量操作不工作？
**A**:
1. 确保已选中案件（勾选复选框）
2. "批量操作"按钮才会显示
3. 选择操作后按提示完成

### Q5: 系统链接点击无效？
**A**: 所有链接已修复（路由完整性100%）。如果还是不工作，请清除浏览器缓存后重试。
