# 关键问题修复报告

> **修复时间**: 2026-04-19 20:45
> **修复内容**: 归档页面"查看详情"按钮路径错误

## 问题

用户在归档页面点击"查看详情"按钮时，无法正确跳转到案件详情页。

## 原因

`D:\ZGAI\frontend\src\views\case\archive.vue` 第112行使用了错误路径：

```javascript
window.location.href = `/#/cases/${row.id}`  // ❌ 错误
```

路由配置中是 `/case/:id` 而不是 `/cases/:id`，导致跳转到404页面。

## 修复

```javascript
window.location.href = `/#/case/${row.id}`  // ✅ 正确
```

## 验证

路由配置（`D:\ZGAI\frontend\src\router\index.js` 第40-108行）：
- `/case/list` - 案件列表
- `/case/create` - 新建案件
- `/case/:id` - 案件详情
- `/case/archive` - 归档库
- `/case/trash` - 回收站

## 影响

修复后，归档页面和回收站页面的"查看详情"按钮可以正确跳转到案件详情页。
