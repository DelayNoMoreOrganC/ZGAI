### 2. 统计卡片 - 蓝色渐变风格

**优化前**：
- 纯色图标背景
- 平面设计

**优化后**：
- 彩色渐变图标背景
- 顶部蓝色渐变条装饰
- 增强的阴影效果
- hover上浮动画

**配色方案**：
```scss

  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); // 蓝紫
}

&.stat-success .stat-icon {
  background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%); // 粉红
}

&.stat-info .stat-icon {
  background: linear-gradient(135deg, #a1c4fd 0%, #c2e9fb 100%); // 天蓝
}
```

**数值颜色**：
- 使用 #1890ff 蓝色作为主色
- 与Alpha法律系统风格对齐
