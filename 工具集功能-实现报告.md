## 实现内容

### 1. 诉讼费计算器

**功能**: 根据《诉讼费用交纳办法》计算案件受理费

**支持的案件类型**:
- 财产案件：分段累计计算
  - 不超过1万元：50元
  - 1万-10万元：2.5%
  - 10万-20万元：2.2%
  - 20万-50万元：1.9%
  - 50万-100万元：1.5%
  - 超过100万元：1%
- 离婚案件：50-300元
- 人格权案件：100-500元
- 劳动争议：10元/件

**计算逻辑**:
```javascript
if (amount <= 10000) {
  fee = 50
} else if (amount <= 100000) {
  fee = 50 + (amount - 10000) * 0.025
} else if (amount <= 200000) {
  fee = 2300 + (amount - 100000) * 0.022
} else if (amount <= 500000) {
  fee = 3800 + (amount - 200000) * 0.019
} else if (amount <= 1000000) {
  fee = 6800 + (amount - 500000) * 0.015
} else {
  fee = 11800 + (amount - 1000000) * 0.01
}
```

### 2. 利息计算器

**功能**: 计算借款利息、存款利息等

**计算公式**: `利息 = 本金 × 年利率 ÷ 365 × 天数`

**输入参数**:
- 本金（元）
- 年利率（%）
- 天数

**示例**:
- 本金 100,000 元，年利率 4.35%，365天
- 利息 = 100,000 × 4.35% ÷ 365 × 365 = 4,350 元

### 3. 时效计算器

**功能**: 计算诉讼时效截止日期

**支持的时效类型**:
- 普通诉讼时效：3年（《民法典》第188条）
- 短期时效：1年
- 长期时效：20年（最长保护期）
- 特殊时效：4年（国际货物买卖、技术进出口等）

**计算逻辑**:
```javascript
const endDate = new Date(startDate)
endDate.setFullYear(endDate.getFullYear() + years)
```
