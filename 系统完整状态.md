## 技术栈

### 前端
- Vue 3 + Composition API
- Element Plus UI
- Vite构建工具
- Vue Router路由

### 后端
- Spring Boot 2.7.18
- Spring Security + JWT
- H2数据库（开发）/ MySQL（生产）
- JPA + Hibernate

### AI
- Ollama本地AI（qwen3:8b）
- RAG知识检索（TF-IDF）
- 向量数据库架构（Qdrant，待部署）
