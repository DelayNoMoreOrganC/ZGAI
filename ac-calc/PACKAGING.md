## 已实现功能

### 1. 核心计算
- 上传银行流水（Excel/PDF）自动提取还款记录
- 本金、利率（固定/浮动LPR+BP）、放款日、到期日、加速到期日、计算截止日
- 按月分段计息（每月21日→次月20日）
- 利率调整按调整日当天生效
- 利息/复利计算（利息余额按利率计复利）
- 还款冲销按优先级（费用>罚息>复利>利息>本金）

### 2. 分段还本计划（v0.3新增）
- 多段还本计划，每段可独立设置：生效日、还本方式、频率、金额/比例
- 支持补充协议AI解析 → 自动生成新计划段
- 支持段间切换（原计划 → 补充协议计划）
- 逾期罚息基于分段计划的预期还本日期自动计算

### 3. Excel导出（4个Sheet）
| Sheet | 说明 |
|-------|------|
| 债权汇总 | 案件信息 + 本金/利息/罚息/复利/总债权 + 还款统计 |
| 本息计算表 | 19列完整表格（法院呈交格式），含应还本金/已还本金/累计逾期本金 |
| 还款记录 | 还款日期、类型、金额、分配明细 |
| 计算说明 | 每期公式核对用 |

### 4. 补充协议AI解析（可选）
- 调用Ollama + qwen3:8b 解析自然语言描述
- AI不可用时自动降级，不影响其他功能
