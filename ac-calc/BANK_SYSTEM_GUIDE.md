# 🏦 银行对账单智能识别系统 - 使用指南

## 📋 系统简介

本系统支持**3种银行对账单**的智能识别与债权计算：

| 银行 | 代码 | 支持格式 | 特色功能 |
|------|------|---------|---------|
| 🏛️ **建设银行** | CCB | PDF, Excel | 标准对账单处理 |
| 🦁 **招商银行** | CMB | PDF, Excel | 企业网银对账单 |
| 🌾 **南海农商银行** | NHRCB | PDF, Excel | 贷款流水台账 |

---

## 🚀 快速开始

### 方式1：双击启动（推荐）

**Windows用户**:
```
双击 start_banks.bat
```

**Linux/Mac用户**:
```bash
chmod +x start_banks.sh
./start_banks.sh
```

### 方式2：命令行启动

```bash
streamlit run app_bank_selector.py
```

## 📱 使用流程

### 1. 银行选择页面

启动后，你会看到3个银行卡片：

```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│  🏛️ 建设银行  │  │  🦁 招商银行  │  │  🌾 南海农商  │
│             │  │             │  │             │
│  [点击进入]  │  │  [点击进入]  │  │  [点击进入]  │
└─────────────┘  └─────────────┘  └─────────────┘
```

点击对应的银行卡片进入处理页面。

### 2. 上传对账单

在银行处理页面，上传你的对账单文件：

- **支持的格式**: PDF、Excel (.xlsx, .xls)
- **文件大小**: 建议不超过 50MB

### 3. 自动识别

系统会自动：
- ✅ 识别银行类型
- ✅ 提取关键信息（贷款金额、利率、日期等）
- ✅ 解析还款记录
- ✅ 执行债权计算

### 4. 查看结果

系统会生成：
- 📊 **债权汇总表**: 总览数据
- 📋 **计息明细表**: 分段计息详情
- 💰 **还款记录表**: 还款冲销明细

### 5. 导出Excel

点击下载按钮，保存标准格式Excel文件。

## 🔧 配置说明

### 添加新银行模板

如果需要添加新的银行模板：

1. 在 `data/bank_templates/` 下创建新目录
2. 创建模板配置文件（参考现有模板）
3. 在 `registry.yaml` 中注册

### 修改现有模板

每个银行的配置文件：
- `data/bank_templates/CCB/v1.0.yaml` - 建设银行
- `data/bank_templates/CMB/v1.0.yaml` - 招商银行
- `data/bank_templates/NHRCB/v1.0.yaml` - 南海农商银行

## ❓ 常见问题

### Q1: 如何启动系统？

**A**: 双击 `start_banks.bat`（Windows）或运行 `./start_banks.sh`（Linux/Mac）

### Q2: 支持哪些文件格式？

**A**: 支持 PDF 和 Excel (.xlsx, .xls) 格式

### Q3: 可以自动识别银行吗？

**A**: 目前需要在选择页面手动选择银行，后续版本会支持自动识别

### Q4: 计算结果准确吗？

**A**: 系统会显示提取的数据供您核对，您可以在计算前修改任何数据

### Q5: 如何添加新的银行？

**A**: 参考 `BANK_SYSTEM_DESIGN.md` 文档创建新的模板配置

## 📊 数据模型说明

### 核心数据模型

| 模型 | 文件 | 说明 |
|------|------|------|
| BankTemplate | bank_template.py | 银行模板配置 |
| ExtractedData | extracted_data.py | 提取的数据 |
| CalculationResult | calculation_result.py | 计算结果 |

### 字段映射配置

每个银行模板包含：
- `field_mappings`: 字段提取规则
- `calculation_rules`: 计算规则配置
- `output_format`: Excel输出格式
