# 最新债权统计（判决）— 功能规划

基于现有本息债权表生成（银行）功能，新增"基于判决文书统计至最新欠款日"模块。

## 整体布局

两个功能通过页面顶部 Tab 切换：

```
[ 📋 本息债权表（银行） | ⚖️ 最新债权统计（判决） ]
```

各自独立页面，共享同一套计算引擎。

## 新功能流程

### Step 1 — 输入判决信息

判决文书（金融借款合同纠纷）中法院已确认的金额直接作为**起算基数**：

| 字段 | 来源 | 说明 |
|------|------|------|
| 案号 | 判决文书 | (202x)xxx民初xxx号 |
| 当事人 | 判决文书 | 原告/被告 |
| 判决日确认本金余额 | 判决判项 | 法院认定的尚欠本金 |
| 判决日确认欠付利息 | 判决判项 | 暂计至判决日的利息 |
| 判决日确认欠付罚息 | 判决判项 | 如判决有 |
| 判决日确认欠付复利 | 判决判项 | 如判决有 |
| 判决日 | 判决文书 | 作为新一轮计息起算点 |
| 判决后利率标准 | 判决判项 | 通常为：LPR 或 合同约定利率上浮50% |

### Step 2 — 计算参数

| 参数 | 说明 |
|------|------|
| 计算截止日 | 默认今天（最新欠款日） |
| 判决后利率 | 默认从判决书提取，可手工修改 |
| 罚息上浮比例 | 默认50% |
| 计息基准 | 360/365 |
| 是否计复利 | 默认是（按判决） |

### Step 3 — 计算结果

```
判决日余额             新增部分（判决日→截止日）
┌─────────────┐        ┌──────────────────────┐
│ 本金        │  +     │ 本金不新增           │
│ 利息        │  +     │ 利息（按判决后利率）  │
│ 罚息        │  +     │ 罚息（对逾期本金）    │
│ 复利        │  +     │ 复利（对欠息余额）    │
└─────────────┘        └──────────────────────┘
        = 最新总债权（截至计算截止日）
```

### Step 4 — 导出Excel

复用现有导出器，增加"判决债权统计"Sheet 模板。

## 与现有银行功能的关系

| 维度 | 银行模式 | 判决模式 |
|------|---------|---------|
| 输入 | 银行流水Excel/PDF | 判决数据（手动录入） |
| 计息起点 | 放款日 | 判决日 |
| 利率 | 合同约定（浮动） | 判决后利率（通常固定） |
| 还本计划 | 分段还本计划 | 无需（本金余额固定） |
| 罚息 | 基于逾期本金 | 基于判决确认逾期本金 |
| 复利 | 欠息余额计复利 | 按判决 |
| 还款记录 | 从流水提取 | 判决后通常无还款 |
