✅ 开发周期：45分钟
✅ 部署复杂度：低（前端代码编译即可）

## 技术实现

### 前端代码

**文件**: `D:\ZGAI\frontend\src\views\knowledge\rag.vue`

**核心流程**：
```javascript
// 步骤1: 搜索知识库
const { data: searchData } = await request({
  url: '/knowledge/list',
  method: 'get',
  params: { page: 1, size: 5 }
})

// 步骤2: 构造上下文
const context = docs.map((doc, index) =>
  `[文档${index + 1}] ${doc.title}\n${doc.summary}...`
).join('\n')

// 步骤3: 调用AI生成答案
const { data: answer } = await request({
  url: '/ai/chat',
  method: 'post',
  data: { message: `基于以下文档回答问题：\n${context}\n问题：${question}` }
})

// 步骤4: 显示答案 + 来源文档
answer.value = answer
sources.value = docs.slice(0, 3)
```

**路由配置**：
```javascript
{
  path: 'knowledge/rag',
  name: 'KnowledgeRAG',
  component: () => import('@/views/knowledge/rag.vue'),
  meta: { title: 'AI知识问答', icon: '🤖' }
}
```

1. **缓存搜索结果**：
