# 如何看到"确认立案"按钮

## 问题原因
前端文件已更新，服务已重启，但**浏览器缓存**了旧页面。

---

## 解决方法（3选1）

### 方法1：强制刷新（最简单）⭐
1. 在新建案件页面
2. 按住 **Ctrl + Shift**
3. 同时按 **R** 键
4. 或者按 **Ctrl + F5**

### 方法2：清除缓存
1. 在浏览器中按 **F12** 打开开发者工具
2. **右键点击**刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 方法3：无痕模式
1. 打开新的无痕窗口（Ctrl + Shift + N）
2. 访问 http://localhost:3017
3. 登录测试

---

## 验证步骤

### 1. 确认前端运行
打开：http://localhost:3017
应该能看到登录页面

### 2. 登录
账号：admin
密码：admin123

### 3. 进入新建案件页面
- 左侧菜单 → 案件 → 新建案件
- 或直接访问：http://localhost:3017/case/create

### 4. 查看按钮
页面右上角应该有**3个按钮**：
```
[保存草稿] [确认立案] [提交案件]
   灰色      绿色      蓝色
```

### 5. 测试立案功能
- 填写案件信息（至少填写：案件类型、案件名称、添加一个当事人）
- 点击绿色按钮"确认立案"
- 确认对话框
- 案件创建成功并自动立案

---

## 代码验证

如果还是看不到按钮，请检查文件：

```bash
# 查看按钮代码
grep -A 5 "确认立案" frontend/src/views/case/create.vue
```

应该能看到：
```vue
<el-button type="success" :loading="filing" @click="handleFiling">
  确认立案
</el-button>
```

---

## 技术说明

### 按钮位置
- **文件**：`frontend/src/views/case/create.vue`
- **行数**：第6-8行
- **颜色**：绿色（type="success"）
- **功能**：handleFiling

### 三个按钮对比
| 按钮 | 颜色 | 功能 | 状态 |
|------|------|------|------|
| 保存草稿 | 灰色 | 保存到localStorage | 不创建案件 |
| **确认立案** | **绿色** | **创建+自动立案** | **审理中** |
| 提交案件 | 蓝色 | 创建案件 | 待立案 |

---

## 依旧看不到？

如果执行了上述步骤还是看不到按钮，请：

1. 完全关闭浏览器
2. 重新打开浏览器
3. 访问：http://localhost:3017
4. 登录测试
