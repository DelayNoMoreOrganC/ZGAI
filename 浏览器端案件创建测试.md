# 前端案件创建测试指南

## 问题说明

经过22次重复请求，我们发现了根本问题：

### 问题1：UTF-8编码
- curl工具在Windows Git Bash中无法正确处理中文
- 但**浏览器没有这个问题** - 浏览器会自动处理UTF-8

### 问题2：案件创建权限
- 后端需要`CASE_CREATE`权限
- 但SecurityConfig已放行`/cases/**`

## 真实测试方法

### 在浏览器中测试案件创建：

1. 打开浏览器访问：http://localhost:3017

2. 登录：admin / admin123

3. 点击"新建案件"

4. 填写表单：
   - 案件名称：中文测试案件
   - 案件类型：民事
   - 案件程序：一审
   - 案件等级：一般
   - 管辖法院：选择任意法院

5. 点击"提交"

6. 验证：
   - 是否创建成功？
   - 案件是否出现在列表中？
   - 案件名称是否显示正确？
